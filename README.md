# 👨🏻‍💻 About Me

<img align="right" width="300" src="https://raw.githubusercontent.com/sanjay-kv/sanjay-kv/main/Assets/illustration.png"/>

I'm a **.NET Software Engineer with 4+ years of experience** building scalable backend systems and enterprise applications.

I enjoy designing **high-performance APIs, distributed systems and microservices architectures** using modern engineering practices.

My development approach focuses on applying:

* **Domain-Driven Design (DDD)**
* **Clean Architecture**
* **CQRS**
* **Mediator Pattern (MediatR)**

Besides coding, I also share knowledge by building **developer tools, websites and technical ebooks**.
